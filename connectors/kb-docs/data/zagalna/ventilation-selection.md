# Підбір вентилятора за робочою точкою
tags: вентиляція, підбір, продажі

Робоча точка визначається тиском (Па) та продуктивністю (м³/год). Для підбору
моделі вентилятора введіть необхідний тиск і продуктивність, а також щільність
середовища. Система підбирає модель, діаметр робочого колеса та потужність
двигуна під задані параметри.

Якщо потрібного тиску не досягає одна модель — розглядається послідовне
з'єднання або більший типорозмір колеса.
