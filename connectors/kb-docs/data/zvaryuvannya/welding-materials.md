# Зварювальні матеріали та режими
tags: зварювання, матеріали, електроди

Для зварювання вуглецевих сталей застосовують електроди з рутиловим або основним
покриттям. Основне покриття дає шов підвищеної пластичності й ударної в'язкості,
рутилове — стабільну дугу та гарне формування шва.

Діаметр електрода підбирають за товщиною металу: для 3–5 мм — електрод 3 мм,
для 6–10 мм — електрод 4 мм. Зварювальний струм орієнтовно 30–40 А на кожен
міліметр діаметра електрода.

Перед зварюванням вологі електроди обов'язково прожарюють згідно з режимом
виробника, інакше у шві з'являються пори.
